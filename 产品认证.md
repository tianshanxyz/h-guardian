# 产品认证

## 认证总览

H-Guardian 拥有完整的国际认证体系，产品通过CE、FDA、SGS、ISO等多项国际权威认证，满足全球不同市场的准入要求。

---

## 国际认证

### 1. CE 认证 (欧盟)

| 认证项目 | 详情 |
|----------|------|
| **认证类型** | CE 标志 (European Conformity) |
| **适用法规** | 欧盟医疗器械法规 MDR 2017/745 |
| **产品分类** | I类医疗器械 |
| **认证编号** | CE-MDR-2017-745 |
| **认证机构** | 欧盟公告机构 (Notified Body) |

#### CE 认证测试项目

| 测试项目 | 标准要求 | 我们的实测值 |
|----------|----------|--------------|
| 细菌过滤效率 (BFE) | ≥95% | ≥99% |
| 呼吸阻力 | <40 Pa | 合格 |
| 微生物清洁度 | ≤30 cfu/g | 合格 |
| 合成血液穿透 | 通过 (16.0 kPa) | 通过 |

#### CE 认证标志含义

- 产品符合欧盟安全、健康和环境保护要求
- 可在欧盟27个成员国自由流通
- 产品可标注CE标志
